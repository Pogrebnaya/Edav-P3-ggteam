point_data_NYCRmd.html : Details of point data information of NYC


park.csv : Park information in NYC, source is Google Places API
restaurant.csv : Restaurant information in NYC, source is Google Places API
police.csv : Police station information in NYC, source is http://www.nyc.gov/html/nypd/html/home/precincts.shtml
school.csv : School information in NYC, source is https://data.cityofnewyork.us/Education/School-Point-Locations/jfju-ynrr

Each csv has name, lat, lng, and other information.
